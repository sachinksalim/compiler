# Compiler Assignment 1
## Lexer for Javascript using PLY

A lexer for Javascript using the parsing tool PLY (Python Lex-Yacc). It outputs a summary of the tokens in the program.


### How to run

```
cd asgn1
make
bin/lexer test/test1.js
```
